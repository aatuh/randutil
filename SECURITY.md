# Security Policy

## Supported versions

This project supports the latest release on the default branch. Older
versions may not receive security fixes.

## Reporting a vulnerability

Please report security issues via GitHub Security Advisories:
https://github.com/aatuh/randutil/security/advisories

Useful private reports include the affected version or commit, a minimal
reproducer, expected and actual behavior, and the security impact. Do not put
secret seeds, generated tokens, private entropy, or exploit details in public
issues.

If that is not possible, open a minimal public issue without sensitive
details and request a private channel.
